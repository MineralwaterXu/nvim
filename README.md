# Neovim
This is my Neovim configuration.
## Keyboard Shortcuts
### 1 Basics
**`i`** : switch to **`INSERT`** : mode

**`Q`** : quit current Neovim window (same as command `:q` in default Settings)

**`S`** : save current file (same as command `:w` in default Settings)
